You can run the file as is and load the functiuons. Or if you want to automate it save it as 

Microsoft.PowerShell_profile.ps1

file path location 
C:\Users\YourUserName\Documents\PowerShell\Microsoft.PowerShell_profile.ps1

It will be loaded with your powershell environment automatically when you launch pwsh in windows. Depending on how you set up VSCode powershell you coiuld have several versions. 
For VSCode you can save it as C:\Users\YourUserName\Documents\PowerShell\Microsoft.VSCode_profile.ps1
